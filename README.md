# Moon OS

**The World's Best Gaming-Focused Linux Operating System**

Moon OS is a performance-optimized, gaming-focused Linux distribution designed to rival Windows in gaming performance while providing a macOS-level polish experience. Built from the ground up for gamers, by gamers.

## Features

### Gaming Performance
- **Custom Gaming Kernel** - Optimized for low latency and maximum throughput
- **GPU Auto-Detection** - Automatic NVIDIA, AMD, Intel GPU configuration
- **GameMode Integration** - Automatic performance tuning during gaming
- **MangoHud** - Real-time performance monitoring overlay
- **Proton/Wine** - Seamless Windows game compatibility
- **HDR Support** - Full HDR10 support for compatible displays
- **VRR Support** - Variable Refresh Rate (FreeSync/G-Sync)
- **Low Latency Audio** - PipeWire with gaming-optimized configuration
- **Input Lag Reduction** - Optimized USB polling and input processing

### Hardware Support
- **Intel CPUs** - Full support with Intel PowerTOP optimization
- **AMD CPUs** - Full support with AMD P-State driver
- **NVIDIA GPUs** - Automatic driver installation (550+)
- **AMD GPUs** - Mesa with RADV and AMDGPU
- **Intel GPUs** - Mesa with ANV
- **Laptops** - Automatic power management and suspend/resume
- **Desktops** - Maximum performance profiles
- **Mini PCs** - Optimized for compact form factors
- **Gaming Handhelds** - Steam Deck, ROG Ally, Legion Go support
- **VirtualBox** - Guest additions included
- **VMware** - Open VM Tools included
- **KVM/QEMU** - VirtIO drivers included
- **Hyper-V** - Integration services included

### Boot Options
- **Legacy BIOS** - Full support
- **UEFI** - Full support with Secure Boot
- **Secure Boot** - Signed bootloader and kernel
- **Dual Boot** - Automatic Windows detection
- **Live USB** - Full desktop experience
- **Persistent USB** - Save your data across sessions

## Installation

### System Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 64-bit x86_64, 2 cores | 8+ cores, 4GHz+ |
| RAM | 4 GB | 16+ GB |
| Storage | 20 GB | 100+ GB SSD |
| GPU | DirectX 11 compatible | NVIDIA RTX 3060+ / AMD RX 6600+ |
| Network | Broadband Internet | Gigabit Ethernet |

### Installation Media

Download the latest ISO from [moonos.dev](https://moonos.dev/download)

### Installation Options

1. **Live USB** - Boot and try without installing
2. **Install** - Full installation to disk
3. **Dual Boot** - Install alongside Windows
4. **Persistent USB** - Install to USB with persistence

### Quick Start

1. Download the ISO
2. Create a bootable USB using Rufus, Etcher, or dd
3. Boot from USB
4. Follow the installer
5. Start gaming!
